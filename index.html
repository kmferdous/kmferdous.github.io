Thesis

  • Title: An Empirical Analysis of LLM-Based Agentic Approaches for Breaking Change Detection in Python Open Source Packages

    Master’s Thesis (ongoing), Department of Computer Science, Kennesaw State University, USA.

    The objective of this research is to investigate the capability of Large Language Model (LLM)-based AI agents in detecting breaking changes within Python open-source software packages. The study focuses on understanding how autonomous agents reason over code changes, analyze commit-level patches, and identify structural and semantic breaking changes introduced during software evolution.

    We design and implement an agentic framework that integrates multiple LLMs as reasoning engines, combined with program analysis techniques. The agent is responsible for identifying affected files, classifying breaking change types, and assessing the potential impact on downstream dependencies.

  • Title: Rule Based Machine Translation Between Bangla & English

    Bachelor’s Thesis, Department of Computer Science and Engineering, BUET, Bangladesh. (B.Sc)

    The objective of this research is to find out a parsing grammar that can be used to find out the correctness of sentence, and also to provide some transformation rules so that the sentence in one language can be translated to the sentence in another language in an efficient way. We make the parsing and the construction to the target language at a time, so that when parsing of the base language is completed the translation to the other language is also at the finished line. Using the same grammar, we translate the target language to be translated to the base language.

    We develop the transformation technique in such a way that, the parsing and the transformation will be held at a time. That means when the checking for syntactically correctness for any language ends the transformation also ends. If the input sequence is syntactically correct, the whole output will be the sequence of the target language. We do this parsing and transformation using the CYK algorithm. If the input sequence is syntactically incorrect, then there also be an output, but it will not correct. So, the transformation does not need to look forward whether the parsing result is negative or positive.
